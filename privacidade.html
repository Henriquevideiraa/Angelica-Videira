Política de Privacidade

1. Recolha de Dados

Recolhemos apenas os dados necessários para o funcionamento dos nossos serviços.

2. Utilização dos Dados

Os dados recolhidos são utilizados exclusivamente para fins de contacto e melhoria dos serviços.

3. Proteção dos Dados

Implementamos medidas de segurança para garantir a proteção das informações pessoais.

4. Direitos do Utilizador

O utilizador pode solicitar a consulta, alteração ou eliminação dos seus dados pessoais.
